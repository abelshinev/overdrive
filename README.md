# OVERDRIVE RACING — Team Website

A compact website showcasing RSET's racing team: OVERDRIVE and its members, alumni, team history, events and dedicated gallery.

**Built by the creative team and developers of [Zeyno](https://zeyno.my).**

---

### **What this project is**

- A small React + TypeScript site that lists team members, faculty advisers, and alumni, and provides individual profile pages.
- Includes public assets for alumni photos (public/alumni) and a group photo gallery.

**Tech Stack**

- React
- TypeScript
- Vite (development/build)
- React Router
- Tailwind CSS (utility-first styling)
- Motion (motion/react) for simple animations
- PostCSS / ESLint (project tooling)

Note: exact packages may vary; check `package.json` for concrete versions.

**Project Structure (high level)**

- `src/` — application source
  - `pages/` — page components (e.g., `AlumniPage.tsx`, `TeamSection.tsx`)
  - `components/` — reusable UI components (e.g., `TeamCard`, `Navbar`)
  - `lib/` — data helpers (e.g., `teamMembers.ts`)
- `public/` — static assets
  - `public/alumni/` — moved alumni images and `group/` gallery
